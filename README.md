
## اقفال ملف بايثون بباسوورد

| اسم الملف | وظيفته |
| ------ | ------ |
| tools.py | ذا اللي ترسله للي تبيه يستحدم اداتك |
| foradmin.py | ذا لك لاترسله لاحد |


بيطلع له مفتاح يرسله لك .. كمثال على المفتاح tools.py اللي بيستخدم:

```sh

```
